Highlight #1 What is a VCS

A VCS, or Version Control System, is a system of storing and recalling information. It allows you to store and recall specific versions of a information.

Highlight #2 Git basics

A git is a type of Version Control System that uses a repository to store, manage, and recall files. You can revisit older versions of files and recall them to work on them linearly.

Highlight #3 Git Commands

One of the things we learned in this lesson was how to use the git commands to manage your files and repository. The command git init sets up your respository. The commad git commit adds files to the repository you've named.

Highlight #4 Git Practice

Another point of the lesson was to practice using git repositories by manipulating file versions. The product was called TryGit and was offered by GitHub.

Highlight #5 VCS as a tool

Version Control Systems are tools. Git repositories and GitHub can be a useful tool to store, share, and manage different versions of files over time. Using the git repositories can be useful for many types of files and a very helpful tool.
